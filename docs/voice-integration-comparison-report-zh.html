完整语音交互闭环

说话 → Handy 转录 → agent-meow 接收 → Hermes 执行 → Voicebox 朗读响应。全程本地、全程离线。

sequenceDiagram
    actor User as 用户
    participant Handy as Handy STT
    participant Meow as agent-meow
    participant Hermes as Hermes Agent
    participant Voicebox as Voicebox TTS

    User->>Handy: 按下快捷键, 说话
    Handy->>Handy: VAD + Whisper/Parakeet
    Handy->>Meow: 粘贴转录文本到输入框
    Meow->>Meow: POST /v1/sessions/<id>/events
    Meow->>Hermes: HermesExecutor.run_turn()
    Hermes->>Hermes: computer_use / shell / skills
    Hermes->>Meow: TextChunk + TurnComplete
    Meow->>Meow: 会话流 → SSE → Web UI
    Meow->>Voicebox: speak 工具 → POST /speak
    Voicebox->>Voicebox: TTS 引擎 + 语音配置
    Voicebox->>Meow: audio_url + generation_id
    Meow->>User: AudioBlock 内嵌播放器
